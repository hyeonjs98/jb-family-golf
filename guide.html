1. 앱 소개

JB Family Golf는 우리 가족 골프 모임의 스코어를 기록하고, 개인별 성적 추이와 핸디캡을 관리하는 웹 앱입니다.

주요 기능
- 전체 멤버 스코어 현황 한눈에 보기
- 스코어카드 사진 촬영으로 자동 입력
- 게임 스코어로 핸디캡 대비 실력 비교
- 개인별 통계, 스코어 추이 차트
- 라운드 기록 관리 (수정/삭제)

앱 화면 구성 (탭)

탭설명
스코어 현황전체 멤버의 핸디캡, 최근 스코어, 게임 스코어, 평균, 베스트를 한눈에 비교합니다. 게임 스코어 기준으로 순위가 매겨집니다. 멤버 이름을 탭하면 개인 상세 통계를 볼 수 있습니다.
기록 보기멤버 목록에서 개인을 선택하면 해당 멤버의 라운드 기록 목록이 표시됩니다. 각 라운드를 탭하면 홀별 스코어카드를 확인하고 수정/삭제가 가능합니다.
스코어 입력스코어카드 사진 촬영으로 여러 명을 한번에 입력하거나, 수동으로 한 명씩 입력할 수 있습니다. 골프장을 선택하면 홀별 파가 자동으로 설정됩니다.
⚙ (설정)플레이어 이름/핸디캡 관리, 골프장/코스별 파 관리를 할 수 있습니다.

3. 스코어 입력 방식

핵심: 오버타수만 기록

실제 타수가 아닌 파 대비 오버타수만 기록합니다.

오버타수의미예시 (Par 4)
-1Birdie3타
0Par4타
1Bogey5타
2Double Bogey6타
3Triple Bogey7타
예시) Par 4에서 6타를 쳤다면 → 2 입력
예시) Par 5에서 5타를 쳤다면 → 0 입력
예시) Par 3에서 2타를 쳤다면 → -1 입력

입력된 오버타수는 자동으로 실제 스코어로 변환되어 저장됩니다.
(저장 스코어 = 해당 홀 Par + 오버타수)

4. 스코어카드 사진으로 입력하기

가장 편한 입력 방법입니다. 라운드 후 스코어카드를 사진 찍으면 AI가 자동으로 인식합니다.

스코어카드 작성 규칙

  1. 이름은 한글로 앱에 등록된 이름과 동일하게 적어주세요
    ✓ 정석, 슬기, 상훈, 문정, 제원, 수지, 종민, 지현
    ✗ JS, 정석이, Jung 등 다른 표기 사용 금지
  2. 오버타수만 기록
    각 홀에 실제 타수가 아닌 오버타수를 적어주세요
  3. 또박또박 적기
    AI가 인식할 수 있도록 숫자를 깔끔하게 적어주세요

스코어카드 작성 예시

이름 123456789
정석 102101201
슬기 0-110010-10
상훈 112011210
문정 213121212

입력 순서

  1. 스코어 입력 탭 → 촬영 또는 사진 선택
  2. AI가 인식한 결과 미리보기 확인
  3. 골프장, 코스(전반/후반), 날짜 선택
  4. 인식된 이름이 맞는지 확인 (드롭다운에서 변경 가능)
  5. 스코어가 잘못 인식되었으면 직접 수정
  6. 일괄 저장 버튼 클릭
참고: 등록되지 않은 이름은 자동으로 건너뛰기 됩니다. 우리 멤버가 아닌 동반자의 스코어는 무시됩니다.

5. 수동 입력

사진 인식이 어려운 경우 수동으로도 입력할 수 있습니다.

  1. 스코어 입력 탭 하단 수동 입력 영역
  2. 기록 대상 멤버 선택
  3. 날짜 선택
  4. 골프장 선택 → 코스 자동 선택 → 파 자동 입력
  5. 각 홀에 오버타수 입력 (0 = Par)
  6. 라운드 저장 클릭

수동 입력은 한 명씩 저장합니다.

8. 게임 스코어란?

게임 스코어는 최근 라운드에서 자기 핸디캡 대비 얼마나 잘 쳤는지를 나타냅니다.

게임 스코어 = 최근 스코어 - 핸디캡

게임 스코어의미색상
+5핸디캡보다 5타 더 침 (못 침)파란색
0핸디캡과 동일하게 침초록색
-3핸디캡보다 3타 적게 침 (잘 침)빨간색
예시 1) 핸디캡 20, 최근 스코어 22
→ 게임 스코어 = 22 - 20 = +2 (파란색, 핸디캡보다 2타 더 침)

예시 2) 핸디캡 20, 최근 스코어 18
→ 게임 스코어 = 18 - 20 = -2 (빨간색, 핸디캡보다 2타 적게 침)

예시 3) 핸디캡 16, 최근 스코어 16
→ 게임 스코어 = 16 - 16 = 0 (초록색, 핸디캡 그대로)
핵심: 게임 스코어가 낮을수록(마이너스일수록) 그 라운드에서 잘 친 것입니다.
핸디캡이 다른 멤버끼리도 게임 스코어로 공정하게 비교할 수 있습니다.

9. 기록 수정/삭제

  1. 기록 보기 탭 → 멤버 선택
  2. 수정할 라운드 클릭
  3. 수정 버튼 → 오버타수 수정 → 저장
  4. 삭제가 필요하면 삭제 버튼
